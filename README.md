🌍 RoamIn:
RoamIn is a simple frontend-based travel explorer built using only HTML. It showcases some of the most popular tourist destinations across India, allowing users to explore places either state-wise or through a list of globally recognized landmarks.
This project focuses on presenting travel information in a clean and structured way, making it easy to browse and explore.

✨ Features:
1. 🗺️ Explore by State:
                       View tourist destinations categorized by Indian states
                       Helps users explore region-specific places easily
2. 🌟 Globally Famous Places:
                             Separate section highlighting internationally recognized landmarks
                             Ideal for discovering India’s most iconic destinations
3. 📍 Organized Information:
                           Displays names and locations of major tourist spots
                           Structured layout for better readability
4. 🎯 Simple & Clean UI
                        Built with pure HTML
                        Minimal and easy-to-understand interface

Note:  This is only a frontend model (with Html only)...Backend will be developed and added later for the whole functionality of the website.
